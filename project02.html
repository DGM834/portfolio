Project 02: Pokemon Name Translator

Project Description:

My project is a dictionary that translates Pokemons’ English names into their Japanese names. The goal of this project is to get more familiar with dictionary data structure in Python.

Technologies Used:

I used the Python programming language to build the application.

Features Implemented:

The menu project opens with a welcoming “Welcome!” message for new users and asks them to give a Pokemon name to translate. The available Pokemon names in the dictionary include the first 50 Pokedex entries of the first generation Pokemon. If the user enters in one of these names, the translator will output the Japanese (romanized) name of the Pokemon. There is one hidden help feature for the user to stumble upon. Afterwards, the user is asked if they want to play again and if they agree, they will be asked to give another Pokemon name. If they do not, they will be exited out of the program and be given the warm “Goodbye Tomodachi!” message.

Challenges Faced:

I had much difficulty with knowing when to apply the break and continue keywords and integrating boolean values into my loops and if statements.

Learning and Achievements:

Through this project, I was able to become comfortable with coding with dictionaries and for loops.

In the future, I want to implement for loops that could animate the Welcome sign and the Bye Bye sign. I also want to implement a hidden function that would direct the user to a special surprise.

Conclusion:

This project was a way for me to test how comfortable I am with using dictionaries, dictionary methods, booleans, and advanced loop controls.
