score = 0

# this function will display the question and options and return the user's answer
def quiz(question):
    print(question["question"])
    for option in question["options"]:
        print(option)
    answer = input("Enter your answer choice: ").upper()
    return answer

#main game
for question in questions:
    user_answer = quiz(question)
    if user_answer == question["answer"]:
        print("You are correct!")
        score += 1
    else:
        print("You are wrong. Too bad. Haha")
print("Good job, you ended with a final score of", score, "out of", len(questions))

Technologies Used:

I used the Python programming language to build the application.

Features Implemented:

This project is a multiple choice quiz with 12 questions. Each question and its options, A/B/C/D, are displayed to the user. The user must input their answer choice and if their answer is correct, they will be congratulated and their score will go up by 1. If they are incorrect, they will be told they are wrong and move on to the next question. At the end of the quiz, the program will display the total score to the user.
